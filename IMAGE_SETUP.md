## How to Add Images

1. **Download** images from the Google Drive links above
2. **Rename** them according to the filenames listed
3. **Place** them in the `public/images/` directory
4. **Restart** the development server if it's running

## Temporary Placeholders

Until you add real images, the website will show:
- Broken image icons (which is normal)
- The layout and functionality will still work

The website is fully functional, you just need to add the images to complete it!

## Image Optimization Tips

- Use **JPG** for photos (smaller file size)
- Use **PNG** for logo (supports transparency)
- Compress images before uploading (use tools like TinyPNG)
- Keep file sizes under 500KB each for faster loading
- Use descriptive filenames
